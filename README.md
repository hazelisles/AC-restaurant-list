# 我的餐廳清單
---
一個使用Node.js, Express, handlebars打造的餐廳清單網站。可以在首頁看到餐廳的基本資料與評分，並可進一步點進去看詳細餐廳介紹。在首頁可使用搜尋功能找到特定名稱或類型的餐廳。
